Para executar o programa, basta acessar a pasta código e na linha de comando digitar o comando:

java -jar MonstrosDosDadosMasmorra.jar

É importante lembrar que deverá existir um terminal rodando o arquivo 'servidor.jar' dentro da pasta NetGames,
para que o programa funcione de maneira correta. Por ser um executável, não é necessário que nenhum programa 
seja instalado (isso partindo do pressuposto que o usuário tenha Java e JVM instalados no seu computador).

Essa primeira versão do jogo, infelizmente, não contem os sprites dos monstros. Futuramente, isso deve ser
melhorado pelos desenvolvedores. Todas as outras características (requisitos), foram atendidas.
